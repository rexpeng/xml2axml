# xml2axml
[English](README.md) | 中文

XML与AXML互相转换工具

### 用法

#### xml转axml  
``` shell
java -jar xml2axml-2.1.2.jar e [可读的XML文件.xml] [二进制的AXML文件.xml]
```

#### axml转xml
``` shell
java -jar xml2axml-2.1.2.jar d [二进制的AXML文件.xml] [可读的XML文件.xml]
```
